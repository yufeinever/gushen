# Web 检索补充策略

日期：2026-05-21

## 原则

系统不能只依赖当前本地数据集。需要事件、公告、监管、重大新闻、交易规则核验时，允许使用 Web 检索补充可靠数据源。

## 优先级

1. 本地 AKShare 数据和缓存。
2. 交易所、巨潮资讯、证监会、公司公告等一手来源。
3. 东方财富、证券时报、中国证券报等较可靠财经源。
4. 其他新闻源只作为线索，不直接作为交易依据。

## 可靠来源

- 巨潮资讯：`cninfo.com.cn`
- 上交所：`sse.com.cn`
- 深交所：`szse.cn`
- 北交所：`bse.cn`
- 证监会：`csrc.gov.cn`
- 东方财富：`eastmoney.com`
- 中国证券报：`cs.com.cn`
- 证券时报：`stcn.com`

## 约束

- Web 检索只能补充事实和线索，不能替代结构化数据和回测。
- 检索结果必须记录来源、时间、链接和摘要。
- 对交易结论有影响的事件，优先找公告或监管原文。
- 如果只有新闻线索，没有公告或官方来源，必须标记为“待核验”。
