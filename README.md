# gushen

A 股交易研究与决策辅助项目。

本项目先从“交易额前 100 的高流动性股票池”开始，围绕数据采集、因子研究、回测、风控、复盘和 AI 辅助分析建立可审计流程。初期目标不是自动下单，而是先把研究、验证和资金风险隔离清楚。

## 起步原则

- 只研究 A 股，默认股票池为当日成交额前 100。
- AI 用于信息整理、特征生成、风险解释和复盘，不把 AI 输出直接当作买卖指令。
- 资金账户在系统通过回测、模拟盘、风控检查后再接入。
- 每一次实盘决策都必须留下数据版本、信号原因、风险预算和复盘记录。

## 近期里程碑

1. 建立日线行情、成交额排名、基础财务和公告数据入口。
2. 生成每日交易额前 100 股票池快照。
3. 完成第一版非实盘策略研究：趋势、量价、波动、资金活跃度。
4. 建立回测与模拟盘流程。
5. 通过风控门槛后，再讨论资金账户接入方式。
